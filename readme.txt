CellBall ver1.00

【1.CellBallについて】
　　プレイヤーがプロ野球チームのGMとなって優勝を目指すシミュレーションゲームです。
　　excelで動きます。2003以降のバージョンが必要です。

【2.ゲームの起動、終了について】
　　フォルダ内にあるCellBall.xlsを開くとゲームが起動します。ゲームを終了するときはゲーム画面の[×]ボタンで閉じます。
　　なんらかの不具合でゲーム中の[×]ボタンを押せない場合はexcelファイル自体を終了してください。
　　データの保存は自動ではありません。ゲームのメイン画面の右上のアイコンからデータを保存できます。

【3.初回プレイ時の流れ】
　　Cellballを起動すると自動的にスタート画面が出るので「最初から」を選択してください。
　　ゲームは2013年のオフシーズンから開始します。補強を進めて来季の成績へと繋げてください。
　　なお、ゲームを遊ぶには、マクロを有効にして、「保護されたビュー」を解除する必要があります。
